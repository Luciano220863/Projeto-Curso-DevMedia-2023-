Questão|8424

Estilos para o link

Veja o estilos que serão aplicados em um link:

    • Tamanho da fonte de 18 pixels;
    • Família de fonte Arial;
    • Cor de texto verde;

Abaixo o código CSS válido que corresponde ao que é pedido nos itens acima.

a {

font-size: 18px;

font-family: Arial;

color: green;

}
