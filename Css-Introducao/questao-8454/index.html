Questão|8454

Estilos para a tag span

Marque a alternativa que contém o código correto que aplica os estilos abaixo para a tag span:


  • estilo negrito
  • cor de texto vermelha no padrão rgb
  • sublinhado no elemento

Código css correto que aplica o estilo solicitado

span {

font-weight: bold;

color: rgb(255, 0, 0);

text-decoration: underline;

}
