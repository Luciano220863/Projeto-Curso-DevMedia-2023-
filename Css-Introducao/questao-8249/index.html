Questão - 8249

Analisando afirmações


h2 {

color: blue;

color: blue;

}


Abaixo temos 3 afirmações sobre o código CSS apresentado:

  1. O subtítulo terá uma cor de fundo azul.
  2. Os estilos definidos código CSS funcionarão corretamente.
  3. Foi usado o padrão de cor nominal na propriedade color

h2 {

color: blue;

fontSize: 18px;

}

Afirmação 1 - A afirmativa é falsa, a cor de texto do subtítulo será azul

Afirmação 2 - Os estilos não funcionarão corretamente, pois a segunda propriedade está com o nome incorreto

Afirmação 3 - A afirmação está correta, o padrão de cor usado foi o nominal
