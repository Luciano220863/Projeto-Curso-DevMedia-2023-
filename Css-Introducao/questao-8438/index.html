Questão - 8438

Estilos para o span

Abaixo vemos uma lista de estilos para serem aplicados em um elemento span:

  • Tamanho da fonte de 20 pixels
  • Família de fonte Tahoma
  • Cor de fundo #f2f2f2
  • Cor de texto #777777

Preencha as lacunas para que o código CSS fique de acordo com os itens listados acima:

span{

font-size: 20px;

font-family: Tahoma;

background-color: #f2f2f2;

color: #777777;

}
