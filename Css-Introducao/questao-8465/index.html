Estilos para h1 e p


Abaixo temos duas listas de estilos para os elementos h1 (título) e p (parágrafo):

  1. h1 - Título

    • Tamanho da fonte de 32 pixels
    • Família de fonte Impact
    • Remoção do negrito
    • Alinhamento do texto centralizado
    • Efeito de sublinhado no texto

  2. p - Parágrafo

    • Tamanho da fonte de 24 pixels
    • Família de fonte Verdana
    • Alinhamento do texto justificado

Informe o estilos que corresponda ao que foi pedido nas listas acima.


Abaixo a alternativa que contém os estilos que correspondem ao que foi pedido nas listas acima

h1 {

font-size: 32px;

font-family: Impact;

font-weight: normal;

text-align: center;

text-decoration: underline;

}


}

font-size: 24px;

font-family: Verdana;

text-align: justify;

}
