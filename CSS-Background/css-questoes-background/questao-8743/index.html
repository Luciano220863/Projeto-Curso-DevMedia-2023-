Box-card

Você recebeu a tarefa de criar uma classe chamada 'box-card' que precisa atender aos requisitos abaixo:

  • - Exibir a imagem de fundo 'fundo-card.jpg';
  • - Sua largura deve ser 400px;
  • - Sua altura deve ser 200px;

Complete o código abaixo para cumprir sua tarefa na respectiva ordem informada:

Resposta Código CSS aplicada

<style>

.box-card {

background-image: url(fundo-card.jpg);

width: 400px;

height: 200px;

}

</style>
