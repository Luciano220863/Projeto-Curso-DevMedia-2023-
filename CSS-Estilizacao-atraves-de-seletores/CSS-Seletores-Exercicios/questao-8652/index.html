CSS: Seletores
Estilizando elementos pela classe

Veja o código HTML abaixo:

<h1 class="titulo">Matrix 4: Entenda por que o novo filme de Keanu Reeves fracassou nas bilheterias</h1>

<h2 class="subtitulo">Apesar de ter sido considerado um dos filmes mais esperados do ano passado, Matrix Resurrections sofreu uma grande derrota financeira.</h2>

<p class="paragrafo">Um dos filmes mais aguardados de 2021, Matrix: Resurrections fez sua estreia nos cinemas no fim de dezembro. Com um elenco lotado de estrelas, o longa reviveu personagens como <span class="texto-negrito">Neo</span> (Keanu Reeves) e <span class="texto-negrito">Trinity</span> (Carrie-Anne Moss), apresentando, também, novos rostos para mais uma vez levar às grandes telas a icônica mitologia das irmãs Wachowski - que desta vez contou apenas com a direção de Lana. Entretanto, pode-se dizer que nem tudo saiu como os executivos da Warner esperavam. Se em termos de crítica Resurrections contou com análises mistas, recebendo alguns elogios por conta do refresco dado à franquia, e reprovação a respeito do enredo não justificar a existência de mais um filme, falando de bilheteria, o fracasso consolida (mais) um grande golpe aos cofres da Warner Bros.</p>


Complete o código CSS a seguir utilizando as classes dos elementos para definir seus estilos:
  • - título com 24 pixels de tamanho de fonte e família de fonte Tahoma

  • - subtítulo com 22 pixels de tamanho de fonte e família de fonte Arial

  • - parágrafo com 18 pixels de tamanho, família de fonte Arial e altura da linha de 32 pixels

  • span com estilo negrito

Código CSS

.titulo {

font-size: 24px;

font-family: Tahoma;

}


.subtitulo {

font-size: 22px;

font-family: Arial;

}


.paragrafo {

font-size: 18px;

font-family: Arial;

line-height: 32px;

text-align: justify;


}

.texto-negrito {

font-weight: bold;

}
