<div class="secao-servicos">

<img src="img/logo-rodape.jpg">

<p>Site desenvolvido pela agência mix</p>

</div>


O seguinte estilo foi definido para mudar a aparência das imagens:

div img {

width: 300px;

height: 250px;

}

Porém, precisamos que as imagens tenham os seguintes tamanhos:
  1. Imagens da 1ª div:
    • - largura de 300 pixels

    • - altura de 250 pixels

  2. Imagens da 2ª div
    • - largura de 150 pixels

    • - altura de 75 pixels
Abaixo temos a alternativa que corrige o código CSS anterior.

.secao-servicos img {

width: 300px;

height: 250px;

}


.secao-rodape img {

width: 150px;

height: 75px;

}
