Exercícios - Questão 8664

CSS: Seletores
Estilos pelas classes dos elementos


Observe o código abaixo:

<h1 class="titulo-noticia">Robótica</h1>

<p class="paragrafo-noticia">Robô gigante de 20 metros dá seus primeiros passos no Japão - O Gundam está presente em um parque de Yokohama e, infelizmente, não é ágil o suficiente para nos defender contra monstros</p>


Utilizando as classes dos elementos, defina os estilos para eles na seguinte ordem:

  • O título deve ter a margem inferior de 35 pixels;

  • O parágrafo deve ter a cor do texto rgb (255, 60, 35) e seu alinhamento deve ser justificado.

.titulo-noticia {

margin-bottom: 35px;

}


.paragrafo-noticia {

color: rgb(255, 60, 35);

text-align: justify;

}
