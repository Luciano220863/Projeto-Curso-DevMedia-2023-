CSS: Seletores
Descrevendo o código CSS

Observe o código CSS a seguir:

.secao-destaque .texto {

font-size: 16px;

font-family: Tahoma;

}


.texto {

line-height: 36px;

}

A proposta desse projeto é criar uma página web estilizada e que exibe listas. Para isso vamos usar as linguagens HTML e CSS.

Teremos um arquivo HTML que conterá a estrutura da página da imobiliária e um arquivo CSS que conterá os estilos para mudar a aparência da página.

A opção selecionada foi

Opção 1 Todos os elementos com a classe "texto" contidos dentro de elementos com a classe "secao-destaque" terão a altura da linha do texto modificada.
