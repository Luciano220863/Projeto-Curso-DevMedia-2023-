CSS: Seletores|Estilos para links dentro do parágrafo

Preencha o código CSS abaixo de acordo com os itens abaixo:

  • O estilo deve ser aplicado para todos os links dentro de parágrafos

  • A cor de texto do link deve ser (#785520)

  • A cor de fundo do link deve ser (#e5e5e5)

  • Os links devem ter um espaçamento interno de 4 pixels para todos os lados

Resposta aplicada errada, estudar mais o tópico do desafio

a {

color: #785520;

background-color: #000;

padding: 4px;

}
