Exercícios - Questão 8682

CSS: Seletores|Estilos para o subtítulo e o parágrafo


Veja o código HTML abaixo:

<h1>Cientistas descem a 8km de profundidade na Fossa do Atacama: 'Outro planeta'</h1>

<h2>Após três horas de descida, os oceanógrafos chilenos Osvaldo Ulloa e Rubén Escribano chegaram a um lugar nunca visto pelo ser humano.</h2>

<p>Durante anos, os oceanógrafos chilenos Osvaldo Ulloa e Rubén Escribano imaginaram em suas conversas como seria a paisagem "alienígena" da Fossa do Atacama, a fenda impressionante que cai a mais de 8.000 metros de profundidade nas costas do Chile e do Peru, e que nenhum ser humano tinha visto diretamente.</p>


Preencha o código CSS para que o subtítulo e o parágrafo tenham os seguintes estilos em comum:
  • Cor de texto: rgb (45, 50, 78)

  • Texto alinhado à direita

  • Margem superior de 30 pixels
Código CSS que acata as definições indicadas.

h2, p {

color: rgb(45, 50, 78);

text-align: right;

margin-top: 30px;

}
