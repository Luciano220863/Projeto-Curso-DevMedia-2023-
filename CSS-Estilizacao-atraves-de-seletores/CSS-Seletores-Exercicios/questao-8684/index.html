Exercícios - Questão 8684

CSS: Seletores
Estilos para o título


Abaixo temos o elemento título no HTML:

<h1>Vagas para programação</h1>

Preencha no código CSS os respectivos estilos abaixo para o título:

  • Borda esquerda de 3 pixels, tracejada na cor (#787878)

  • Espaçamento interno do lado esquerdo de 5 pixels

  • Cor de texto (#ffffff)

  • Cor de fundo (#454545)
