Exercícios - Questão 8675

CSS: Seletores
Definindo estilos para os elementos

Considere o código HTML abaixo:

<div class="secao secao-destaque">

<h3>Esportes</h3>

<p>Cada vez mais os esportes tomam os destaques das notícias. Fique por dentro de tudo o que ocorre nesse meio aqui no nosso portal.</p>

</div>

<div class="secao">

<h3>Entretenimento</h3>

<p>Memes, assuntos do dia e tudo que gera diversão você encontra aqui.</p>

</div>

<div class="secao">

<h3>Economia</h3>

<p>Todos os assuntos relacionados à Economia você encontra aqui no nosso portal.</p>

</div>


Complete o código CSS atendendo aos itens abaixo:

  • - Utilize apenas as classes dos elementos;

  • - Para todas as divs defina os respectivos estilos:

  • - espaçamento interno de 30 pixels para todos os lados;

  • - margem superior de 20 pixels;

  • - Para a 1ª div defina o seguinte estilo:

  • - Cor de fundo (#e4e7d9);

Esportes

Cada vez mais os esportes tomam os destaques das notícias. Fique por dentro de tudo o que ocorre nesse meio aqui no nosso portal.

Entretenimento

Memes, assuntos do dia e tudo que gera diversão você encontra aqui.

Economia

Todos os assuntos relacionados à Economia você encontra aqui no nosso portal.



Resposta aplicada


<style>

.secao {

padding: 30px;

margin-top: 20px;

}


.secao-destaque {

background-color: #e4e7d9;

}

</style>
