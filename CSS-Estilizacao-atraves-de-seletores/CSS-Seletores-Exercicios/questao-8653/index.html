Exercícios - Questão 8653

CSS: Seletores|Estilos para a imagem

Abaixo temos o código HTML de uma imagem:

<img src="./../img/imovel.jpeg" alt="imovel">

Preencha as lacunas a seguir para que os respectivos estilos sejam definidos na imagem:
  • altura de 100 pixels

  • largura de 250 pixels

  • borda de 4 pixels, pontilhada, na cor #757575

img {

height: 100px;

width: 250px;

border: 4px dotted #757575;

}
