CSS: Seletores
Estilização de span


Abaixo temos o código HTML de um parágrafo:

A <span class="texto-destaque">Revolução</span> Francesa é o nome dado ao ciclo revolucionário que aconteceu na França entre 1789 e 1799 que marcou o fim do absolutismo nesse país. Essa revolução, além de seu caráter burguês, teve uma grande participação popular e atingiu um alto grau de radicalismo, uma vez que a situação do povo francês era precária em virtude da crise que o país enfrentava.


.texto-destaque {

color: #454545;

text-decoration: underline;

}

CSS: Seletores|Questão 8654

Estilização de span

Abaixo temos o código HTML de um parágrafo:


A Revolução Francesa é o nome dado ao ciclo revolucionário que aconteceu na França entre 1789 e 1799 que marcou o fim do absolutismo nesse país. Essa revolução, além de seu caráter burguês, teve uma grande participação popular e atingiu u alto grau de radicalismo, uma vez que a situação do povo francês era precária em virtude da crise que o país enfrentava.

Complete o código CSS utilizando a combinação de seletores para que o elemento span seja estilizado, conforme os seguintes estilos:

  • cor do texto (#454545)
  • estilo sublinhado

Resposta CSS aplicada

<style>

p span{

color: #454545;

text-decoration: underline;

}

</style>
