Exercícios - Questão 8669

CSS: Seletores
Completando os códigos HTML e CSS

Complete os códigos HTML e CSS de acordo com os itens a seguir:

<style>

div {

border: 1px solid #d5d5d5;

background-color: #e8e8e8;

width: 500px;

margin: 15px auto;

font-family: "Tahoma";

}


div h1 {

text-align: center;

color: #454545;

}


div h2 {

text-align: center;

color: #c5c5c5;

}


div h3, div h4 {

color: #454545;

text-align: center;

}


div p {

text-align: justify;

color: #787878;

}


div a {

color: #454545;

text-decoration: none;

}

</style>

A única alternativa que faz uma afirmação falsa sobre o código acima é que indica que:

Todos os subtítulos possuem a mesma cor de texto e o alinhamento centralizado.
