Exercícios - Questão 8666

CSS: Seletores
Estilizando os elementos pela classe

Veja o código HTML abaixo:

<img class="imagem-produto" src="img/bicicleta.jpg" alt="Bicicleta">

A bicicleta está na memória de gerações e gerações de Brasileiros.

Complete o código CSS de modo que a lista de itens seja atendida.

  1. Para estilizar a imagem defina os estilos utilizando a classe do elemento:
    • altura de 350 pixels;

    • largura de 400 pixels;

  2. Já para o parágrafo, defina os seguintes estilos com a classe do elemento:
    • tamanho de fonte de 16 pixels;

    • cor de texto (#ff4455);

Resposta código CSS style aplicada

.imagem-produto{

height: 350px;

width: 400px;

}


.texto-descricao{

font-size: 16px;

color: #ff4455;

}

Resultado Final do código CSS aplicado

A bicicleta está na memória de gerações e gerações de Brasileiros.
