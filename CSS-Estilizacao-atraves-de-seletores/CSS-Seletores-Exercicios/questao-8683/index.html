Exercícios - Questão 8683

CSS: Seletores
Completando os códigos HTML e CSS


Complete os códigos:
HTML e CSS de acordo com os itens a seguir:

  • O elemento título deve ter a classe "titulo";

  • O elemento parágrafo deve ter a classe "paragrafo";

  • Os estilos devem ser aplicados nos elementos pelas suas classes;



  1. Os estilos do título:
    • Tamanho de fonte: 28 pixels

    • Cor do texto: #454545

  2. Os estilo do parágrafo:
    • Tamanho da fonte: 20 pixels

    • Alinhamento do texto: justificado

OBS.: Considere a ordem em que os estilos CSS foram declarados

Abaixo resposta aplicada

.titulo {

font-size: 28px;

color: #454545;

}


.paragrafo {

font-size: 20px;

text-align: justify;

}
