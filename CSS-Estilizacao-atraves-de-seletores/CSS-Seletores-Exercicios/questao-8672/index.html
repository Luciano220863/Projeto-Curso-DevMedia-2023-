CSS: Seletores
Estilizando o elemento span dentro do parágrafo

Veja o código HTML a seguir:

<p>Dos três mesa-tenista do Brasil, Karina foi a única que conseguiu uma vitória nas eliminatórias desta quarta-feira. Pela fase de 64 do campeonato em solo francês, a atleta encarou a espanhola <span>Renata Shypsha</span> e triunfou por 3 sets a 2. No compromisso seguinte, ela não conseguiu repetir o resultado positivo ao ser batida por 3 a 0 para Enisa Sadikovic, de Luxemburgo.</p>

<span>Dezembro de 2021</span>
Complete o código CSS abaixo para que apenas o span dentro do parágrafo receba os respectivos estilos:
  • cor de texto rgb (0,25,125)

  • sublinhado

  • margem esquerda de 5 pixels

  • margem direita de 5 pixels

Dos três mesa-tenista do Brasil, Karina foi a única que conseguiu uma vitória nas eliminatórias desta quarta-feira. Pela fase de 64 do campeonato em solo francês, a atleta encarou a espanhola Renata Shypsha e triunfou por 3 sets a 2. No compromisso seguinte, ela não conseguiu repetir o resultado positivo ao ser batida por 3 a 0 para Enisa Sadikovic, de Luxemburgo.

Dezembro de 2021
A codificação CSS aplicada ao texto acima, é o que segue abaixo

<style>

p span {

color: rgb(0, 25, 125);

text-decoration: underline;

margin-left: 5px;

margin-right: 5px;

}

</style>
