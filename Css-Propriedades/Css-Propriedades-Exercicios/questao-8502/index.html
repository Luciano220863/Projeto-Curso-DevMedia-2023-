Estilos para imagem


Indique a codificação CSS correta que atenda a lista de estilo abaixo para serem aplicados em uma imagem:

  • Largura de 300 pixels
  • Altura de 500 pixels
  • Margem superior de 25 pixels


Veja abaixo o código CSS que corresponde exatamente à lista de estilos acima?

img {

width: 300px;

height: 500px;

margin-top: 25px;

}

}
