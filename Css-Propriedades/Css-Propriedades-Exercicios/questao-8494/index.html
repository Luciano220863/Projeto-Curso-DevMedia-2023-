Estilos para imagem


A seguir vemos uma lista de estilos para serem aplicados em uma imagem:

  • Largura de 800 pixels
  • Altura de 400 pixels
  • Margem superior de 15 pixels

Complete o código CSS para que os estilos sejam aplicados conforme a lista acima:


img {

width: 800px;

height: 400px;

margin-top: 15px;

}
