Estilizando título e parágrafo


Veja os estilos para os elementos abaixo:

  1. Título (h1)
    • Tamanho da fonte de 48 pixels
    • Família de fonte Impact
    • Texto sublinhado
  2. Parágrafo (p)
    • Tamanho da fonte de 28 pixels
    • Cor de texto #565656

h1 {

font-size: 48px;

font-family: 'impact';

text-decoration: underline;

}

p {

font-size: 28px;

color: #565656;

}
