Título e subtítulo


Abaixo vemos alguns estilos que devem ser aplicados para um título (h1) e um subtítulo (h2):

  1. Título:
    • Tamanho de fonte: 28 pixels
    • Família de fonte: Arial
    • Remoção de sublinhado
    • Margem inferior de 20 pixels
    • Cor de texto: #776757
  2. Subtítulo:
    • Tamanho de fonte: 22 pixels
    • Família de fonte: Arial
    • Remoção de negrito
    • Sublinhado


Veja abaixo o código CSS que corresponde exatamente à lista de estilos acima?

h1{

font-size: 28px;

font-family: Arial;

text-decoration: none;

margin-bottom: 20px;

color: #776757;

}


h2{

font-size: 22px;

font-family: Arial;

font-weight: normal;

text-decoration: underline;

}
