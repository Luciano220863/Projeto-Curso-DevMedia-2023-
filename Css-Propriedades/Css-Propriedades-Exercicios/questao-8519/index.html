Mudando a aparência do span


Abaixo temos uma lista de estilos que devem ser aplicados no elemento span::

  • tamanho de fonte de 16px;
  • Negrito;
  • Cor de fundo #e8e8e8;
  • espaçamento interno de 5 pixels para todos os lados;

Complete o CSS abaixo de forma que o código corresponda aos itens listados acima:

RESPOSTA ENVIADA

span {

font-size: 16px;

font-weight: bold;

background-color: #e8e8e8;

padding: 5px;

}

OBS.: Considere a ordem em que os estilos foram definidos.
