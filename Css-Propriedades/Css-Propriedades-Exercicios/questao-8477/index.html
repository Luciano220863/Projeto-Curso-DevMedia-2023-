Definindo estilos para o título


Veja o código abaixo:

<!DOCTYPE HTML>

<html>

<head>

<meta charset="UTF-8">

<title>Tecnologias Front-End</title>

<link rel="stylesheet" href="estilo.css">

</head>

<body>

<div>

<h1>Evento debate acesso a testes genéticos e medicina de precisão</h1>

<p>Estima-se que 6% a 8% da população mundial de 7,8 bilhões de pessoas são afetadas por alguma doença genética, sendo que ao menos 4 entre 10 pacientes recebem um diagnóstico errado ao menos uma vez, complicando seus quadros clínicos.
Por meio da Medicina Genética, dispõe-se de conhecimento para oferecer agilidade, personalização e precisão no diagnóstico e tratamento dos pacientes diagnosticados com estas doenças.
Este contexto também se aplica para as chamadas síndromes hereditárias de câncer.
Por sua vez, é um conhecimento que ainda não chegou a uma grande parcela dos profissionais de saúde e não é realidade para os pacientes no SUS.</p>

</div>

</body>

</html>


Abaixo há uma lista de estilos para mudar a aparência do elemento h1. Veja os itens abaixo:

  • 1º item: Tamanho da fonte de 28 pixels;
  • 2º item: Família de Fonte Arial;
  • 3º item: Cor da fonte azul (padrão hexadecimal com o valor #0000ff);
  • 4: item: Alinhamento do texto centralizado;
  • 5º item: Margem superior de 25 pixels (utilize a propriedade margin-top).

Preencha o código CSS de modo que atenda aos requisitos listados:


Resposta aplicada

h1 {

font-size: 28px;

font-family: 'Arial';

color: #0000ff;

text-align: center;

margin-top: 25px;

}
