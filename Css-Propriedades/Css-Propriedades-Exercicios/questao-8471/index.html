Estilos para o link


Abaixo temos uma lista de estilos que devem ser atendidos para um link:

  • tamanho de fonte de 20 pixels;
  • família de fonte Tahoma;
  • margem de 25 pixels na parte superior do elemento;
  • espaçamento interno de 10 pixels para as laterais do link;
  • espaçamento interno de 5 pixels para as partes superior e inferior do link;
  • cor de texto preta (#222222);
  • cor de fundo cinza (#d4d4d4);

RESPOSTA ENVIADA

a {

font-size: 20px;

font-family: Tahoma;

margin-top: 25px;

padding: 5px 10px;

color: #222222;

background-color: #d4d4d4;

}
