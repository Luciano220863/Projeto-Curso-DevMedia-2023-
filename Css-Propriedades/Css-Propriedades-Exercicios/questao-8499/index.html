Estilos para elemento h4


Seguem abaixo alguns estilos para o elemento h4:

  • - Borda inferior tracejada de 2 pixels na cor rgb (244, 244, 150)
  • - Tamanho da fonte de 20 pixels
  • - Família de fonte Arial
  • - Sublinhado
  • - Alinhamento de texto centralizado

h4 {

border-bottom: dashed 2px rgb(244, 244, 150);

font-size: 20px;

font-family: 'Arial';

text-decoration: underline;

text-align: center;

}
