Estilizando o link


Complete o código CSS para que o elemento <a> (link) receba os estilos de acordo com a lista abaixo:

  • Cor de fundo azul (utilize o padrão de nominal);
  • Cor do texto branca (utilize o padrão de nominal);
  • Espaçamento interno de 10 pixels para todos os lados;
  • Margem superior de 15 pixels (utilize a propriedade específica para a margem superior);
  • Borda inferior de 3 pixels, estilo sólido e cor preta (utilize o padrão de nominal).

Código CSS

a {

background-color: blue;

color: white;

padding: 10px;

margin-top: 15px;

border-bottom: 3px solid black;

}
