Propriedades específicas de padding


Complete o código CSS utilizando propriedades específicas de margem para o elemento <div>:

  • espaçamento interno na parte superior de 20 pixels
  • espaçamento interno na parte inferior de 15 pixels
  • espaçamento interno no lado esquerdo de 30 pixels
  • espaçamento interno no lado direito de 20 pixels

obs.: As propriedades devem ser aplicadas na ordem que aparecem.


div{

padding-top: 20px;

padding-bottom: 15px;

padding-left: 30px;

padding-right: 20px;

}
