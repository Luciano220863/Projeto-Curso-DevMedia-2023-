Aplicando estilos no link


Veja o código HTML do link abaixo:

<a href="https://www.devmedia.com.br/javascript">Trilha de JavaScript</a>

A seguir temos uma lista de estilos para o link visto acima. Preencha o código CSS com os estilos seguindo a ordem especificada abaixo:

  1. estilo: espaçamento interno de 10px para todos os lados do link;
  2. estilo: margem de 15px para todos os lados do link;
  3. estilo: cor de fundo verde com o valor hexadecimal #11ee05;
  4. estilo: cor de texto branca com o valor hexadecimal #ffffff;
  5. estilo: remoção do sublinhado que vem por padrão no link.

OBS.: Utilize as propriedades compactas para definir o espaçamento interno e as margens do elemento.


Veja abaixo o código CSS que corresponde exatamente à lista de estilos acima?

a {

padding: 10px;

margin: 15px;

background-color: #11ee05;

color: #ffffff;

text-decoration: none;

}

OBS.: Utilize as propriedades compactas para definir o espaçamento interno e as margens do elemento.
