CSS: Propriedades | Questão:8517

Analisando estilos do link


Complete o código CSS com as propriedades resumidas para margem e espaçamento interno:

  1. Margens:
    • 15 pixels para a parte superior
    • 10 pixels para a parte inferior
    • 20 pixels para o lado esquerdo
    • 30 pixels para o lado direito

  2. Espaçamento interno:
    • 20 pixels para a parte superior
    • 25 pixels para a parte inferior
    • 10 pixels para o lado esquerdo
    • 15 pixels para o lado direito

OBS.: Considere a ordem em que os estilos foram definidos.

div { margin: 15px 10px 20px 30px; padding: 20px 25px 10px 15px; }
Concluída com erro

Resposta errada

div {

margin: 15px 10px 20px 30px;

padding: 20px 25px 10px 15px;

}
