Propriedades de texto para o elemento h1


Abaixo temos estilos para alguns elementos:

<h1>Soluções de Segurança Pública</h1>

O valor da propriedade "text-decoration" do parágrafo está incorreto.


  • Tamanho de fonte de 48 pixels
  • Família de fonte Verdana
  • Alinhamento do texto centralizado
  • Texto sublinhado
  • Altura da linha de 60 pixels

Abaixo, a codificação CSS que aplicamos para atender os estilos listados acima de forma correta


h1 {

font-size: 48px;

font-family: Verdana;

text-align: center;

text-decoration: underline;

line-height: 60px;

}
