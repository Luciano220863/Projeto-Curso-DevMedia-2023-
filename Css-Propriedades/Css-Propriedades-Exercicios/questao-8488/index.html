Estilos para o link


Todos os links de uma página precisam ter os estilos da lista abaixo:

  • Estilo negrito
  • Espaçamento interno: 10 pixels nas partes superior/inferior e 20 pixels nas partes laterais (forma resumida)
  • Remoção de sublinhado
  • Cor de fundo com o valor: #e4e4e4
  • Cor de texto com o valor: #383838
  • Arredondamento das bordas de 15 pixels para todos os lados


Complete o código CSS abaixo para que o elemento link corresponda à lista de estilos.

a{

font-weight: bold;

padding: 10px 20px;

text-decoration: none;

background-color: #e4e4e4;

border-radius: 15px

}
