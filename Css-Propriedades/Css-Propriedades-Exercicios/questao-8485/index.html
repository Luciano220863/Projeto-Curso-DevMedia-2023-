Alterando a aparência do parágrafo


Abaixo temos uma lista de estilos para o elemento p (parágrafo):

  1. Cor de texto (padrão RGB) com os valores
    • 0 para vermelho
    • 10 para verde
    • 20 para azul

  2. OBS.: Siga a ordem que os estilos foram declarados.
    • Alinhamento do texto à esquerda
    • Altura da linha de 36 pixels

Resposta enviada.

p {

color: rgb(0, 10, 20);

text-align: left;

line-height: 36px;

}

Resposta na tela



Veja os estilos abaixo:

div {

width: 200px;

padding: 5px;

border: 8px solid #000;

border-radius: 15px;

}


p {

font-size: 24px;

text-align: center;

}
