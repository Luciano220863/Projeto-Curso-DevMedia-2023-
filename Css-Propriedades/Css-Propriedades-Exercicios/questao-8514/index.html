Lista de estilos


Veja abaixo a lista de estilos para serem aplicados no elemento div:

  • Margem
    • margem superior de 15 pixels
    • margem inferior de 15 pixels
    • margem direita com valor automático
    • margem esquerda com valor automático
  • Espaçamento interno (PADDING)
    • espaçamento interno na parte superior de 20 pixels
    • espaçamento interno na parte inferior de 20 pixels
    • espaçamento interno no lado esquerdo de 20 pixels
    • espaçamento interno no lado direito de 20 pixels
