Estilos para o parágrafo


Complete o código CSS para que o parágrafo tenha os seguintes estilos:

  • - cor de texto #4580f0
  • - Alinhamento justificado
  • - margem de 5 pixels para todos os lados

RESPOSTA ENVIADA

p {

color: #4580f0;

text-align: justify;

margin: 5px

}

OBS.: Considere a ordem em que os estilos foram definidos

RESPOSTA ESTÁ CORRETA!
