Valores de margem


Observe o estilo para a div abaixo:

div {

margin: 10px 15px;

}

Qual das alternativas descreve exatamente os valores de margem especificados no estilo acima?


RESPOSTA ENVIADA

div {

height: 450px;

width: 800px;

border: 2px solid #968558;

margin: 15px;

padding: 35px;

text-align: right;

}

OBS.: Considere a ordem em que os estilos foram definidos.

RESPOSTA ESTÁ CORRETA!
