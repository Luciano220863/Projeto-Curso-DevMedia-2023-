Arredondando as bordas de um link


As bordas de um link precisam ser arredondadas conforme a lista abaixo:

  • Canto superior esquerdo de 25px
  • Canto superior direito de 30px
  • Canto inferior esquerdo de 25px
  • Canto inferior direito de 30px

Complete o código CSS para que os estilos sejam aplicados conforme a lista acima:


img {

width: 800px;

height: 400px;

margin-top: 15px;

}
