CSS: Propriedades | Questão:8487

Bordas e margens

Veja abaixo a lista deles:

  1. Alguns estilos devem ser aplicados para o elemento
    • borda superior sólida de 1 pixel na cor vermelha (#ff2200)
    • borda direita sólida de 1 pixel na cor vermelha (#ff2200)
    • borda inferior pontilhada de 5 pixels na cor preta (#454545)
    • borda esquerda pontilhada de 5 pixels na cor preta (#454545)

    • margem superior de 15 pixels
    • margem inferior de 20 pixels

      2. OBS.: Siga a ordem em que os estilos foram declarados.

      Para cada borda siga a ordem:

    • 1º largura
    • 2º estilo
    • 3º cor

div {

border-top: solid 1px #ff2200;

border-right: solid 1px #ff2200;

border-bottom: dotted 5px #454545;

margin-top: 15px;

margin-bottom: 20px;

}
