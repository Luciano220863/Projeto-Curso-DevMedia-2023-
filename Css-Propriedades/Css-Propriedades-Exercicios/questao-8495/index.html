Estilizando o elemento span


Abaixo temos alguns estilos para serem aplicados no elemento span:

  • Estilo negrito
  • Sublinhado
  • Cor de texto #556900
  • espaçamento interno para todos os lados de 10 pixels

div {

margin: 10px 15px;

}

Qual das alternativas descreve exatamente os valores de margem especificados no estilo acima?


RESPOSTA ENVIADA

span {

font-weight: bold;

text-decoration: underline;

color: #556900;

padding: 10px;

}

OBS.:Siga a ordem que os estilos foram definidos.

RESPOSTA ESTÁ CORRETA!
