Afirmações sobre os estilos


Observe o código CSS a seguir:


h2 {

color: #a1a1a1;

background-color: #000000;

font-weight: bold;

text-decoration: overline;

text-align: center;

padding: 10px;

}


  • Afirmação 1 - Foi aplicado um sublinhado no subtítulo.
  • Afirmação 2 - Uma margem de 10 pixels foi aplicada em todos os lados do subtítulo.
  • Afirmação 3 - O estilo negrito foi aplicado no subtítulo.
