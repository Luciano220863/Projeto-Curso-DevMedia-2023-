Estilos para o parágrafo


A seguir vemos uma lista de estilos que devem ser aplicados em um parágrafo:

  • Largura de 700 pixels
  • Alinhamento justificado
  • Tamanho da fonte de 18 pixels
  • Família de fonte Verdana
  • Altura da linha de 28 pixels para o texto
  • Cor de texto #787878

p{

width: 700px;

text-align: justify;

font-size: 18px;

font-family: Verdana;

line-height: 28px;

color: #787878;

}
