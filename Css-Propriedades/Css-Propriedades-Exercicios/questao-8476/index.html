Trabalhando com bordas no CSS


Abaixo temos um trecho de código HTML que possui uma div que faz parte de um documento HTML:

<h1>Vagas para programadores</h1>

<p>O mercado de TI está mais aquecido do que nunca. Um dos momentos que mais oferecem oportunidades chegou e você não pode ficar de fora dessa!</p>

</div>


Os seguintes estilos são necessários para o container:

  • Borda esquerda de 5 pixels, sólida e azul;
  • Borda superior de 1 pixel, sólida e azul
  • Borda inferior de 1 pixel, sólida e azul

Preencha o código CSS abaixo de acordo com os itens listados.


Em cada borda siga a seguinte ordem das propriedades: tamanho da borda, estilo da borda e cor da borda.

Siga a ordem que os itens foram listados, e para a cor utilize o padrão nominal.
