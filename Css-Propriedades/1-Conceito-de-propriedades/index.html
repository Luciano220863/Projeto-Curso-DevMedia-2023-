1. Conceito de propriedades


No curso de Introdução ao CSS vimos alguns conceitos introdutórios do CSS que servem de base para o aprendizado nessa linguagem.
Agora vamos abordar as propriedades no CSS e como elas alteram a aparência de um elemento. Vamos iniciar relembrando o conceito de propriedades no flow a seguir:



Nos Códigos 1 e 2 seguem os códigos que foram usados no flow.

Código 1. Código HTML

1 | <a href=”#contato”>Contato*lt;/a>


Código 2. Código CSS

1 | a {

2 | font-size: 18px;

3 | color: white;

4 | background-color: red;

5 | }


Na Figura 1 vemos que cada propriedade altera um aspecto visual do elemento. Enquanto uma propriedade altera o tamanho da fonte, temos outras que modificam as cores (do texto e de fundo) do elemento.


As propriedades representam características dos elementos das páginas web. Quando há uma propriedade com um valor definido, temos um estilo do elemento sendo configurado, o que altera sua aparência na página
