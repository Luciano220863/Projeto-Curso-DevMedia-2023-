Questão|8426

Preenchendo os códigos HTML e CSS

Preencha os códigos abaixo de acordo com os itens listados:


  • No arquivo index.html deve ter um link com o valor "#sobre" no atributo href. Na tela o texto dentro do link deve ser "Sobre"
  • No arquivo estilo.css devem ter os seguintes estilos para o link:
  1. º estilo: cor da fonte branca
  2. º estilo: cor de fundo verde
  3. º estilo: tamanho de 14 pixels na fonte

OBS.: Siga a ordem em que os estilos foram listados e utilize o padrão de cor nominal nas cores

Código HTML

<!DOCTYPE HTML>

<html>

<head>

<meta charset="UTF-8">

<title>Propriedades CSS</title>

<link rel="stylesheet" href="estilo.css">

</head>

<body>

Sobre

</body>



OBS.: Siga a ordem em que os estilos foram listados e utilize o padrão de cor nominal nas cores

Código HTML

a {

color: white;

background-color: green;

font-size: 14px;

}
