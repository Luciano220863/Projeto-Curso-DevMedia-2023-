Lista de estilos

Abaixo temos uma lista de estilos que precisa ser aplicada em um parágrafo:

  • tamanho da fonte 18 de pixels
  • cor azul (padrão nominal)
  • altura da linha de 24 pixels
  • text-align: justify;

p {

font-size: 18px;

color: blue;

line-height: 24px;

text-align: justify;

}
