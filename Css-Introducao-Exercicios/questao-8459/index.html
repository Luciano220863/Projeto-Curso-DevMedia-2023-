Estilos para o subtítulo

Abaixo vemos uma lista de estilos para o elemento h2:


  • Tamanho de fonte: 28 pixels;
  • Família de fonte: Verdana;
  • Remoção do negrito, deixando o texto normal (sem utilizar valores numéricos)
  • Cor de texto rgb (255, 245, 235)
  • Alinhamento do texto centralizado

Seguindo a ordem que os estilos foram definidos na lista, preencha as lacunas com os estilos correspondentes:

Código HTML

h2 {

font-size: 28px;

font-family: Verdana;

text-decoration: none;

color: rgb(255, 245, 235)

text-align: right

}
