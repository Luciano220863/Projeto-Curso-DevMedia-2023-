Estilos para o título CSS

Veja a lista de estilos para serem definidos no elemento h1:

    • Cor de texto #787878;
    • Família de fonte Tahoma
    • Alinhamento do texto centralizado
    • Sublinhado no texto

Seguindo a ordem que os estilos foram listados acima, complete as lacunas para que os estilos sejam aplicados no título:

<style>

h1 {

color: '#787878';

text-align: 'justify';

}

link {

text-decoration: 'none';

font-size: '16px';

}

</style>
