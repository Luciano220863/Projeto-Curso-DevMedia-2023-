Questão 8430

Estilização de título


Observe o código HTML abaixo onde temos um container que contém um título e um parágrafo::

<div>

<h1>Os mistérios que cercam os maiores buracos negros do universo</h1>

<p>A estação de transição entre o inverno e o verão começa oficialmente nesta quarta-feira e vai até o dia 21 de dezembro.</p>

</div>



O código CSS para que o título receba os seguintes estilos segue abaixo:

  • Cor de texto #454545
  • Tamanho da fonte de 32 pixels
  • Família de Fonte "Verdana"



h1 {

color: #454545;

font-size: 32px;

font-family: Verdana;
