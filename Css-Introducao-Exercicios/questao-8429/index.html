Completando os códigos HTML e CSS

Complete os códigos HTML e CSS seguindo os itens abaixo:

  • No código HTML:

    • Na 1ª e na 4ª lacuna devem ser usadas tags de abertura e fechamento para o contêiner;
    • Dentro do contêiner há um parágrafo.

  • No código CSS:

    • Aplicar uma cor de fundo preta para o contêiner (padrão de cor nominal);
    • Definir uma fonte de tamanho 18 píxeis para o parágrafo;
    • Definir a cor de texto branca para o parágrafo (padrão de cor nominal).

Código HTML

<div>

<p>Eu sou um desenvolvedor front-end</p>

</div>




Código CSS

div {

background-color: black;

}


p {

font-size: 18px;

color: white;

}
