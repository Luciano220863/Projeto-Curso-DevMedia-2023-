Questão|8466

Estilos para div e h3


Veja o código HTML abaixo:

<div>

<h3>Novidades do dia</h3>

<p>Nessa seção você vai receber notícias recentes sobre os últimos acontecimentos na cidade.</p>

</div>


  • Cor de fundo vermelha para o elemento div;
  • Cor de texto branca para o elemento h3;
  • Alinhamento de texto centralizado para o elemento h3;

div {

color: red;

}

h3 {

color: white;

text-align: center;

}
