Questão|8460

Estilos para as tags p e span

Veja o código HTML abaixo:

<p>Falando ao site Wall Street Journal (via VGC), Phil Spencer, <span>chefe da divisão Xbox na Microsoft</span>, revelou que a companhia não tem planos de apostar no mercado de dispositivos de realidade virtual, apesar de aplaudir o que Sony, Oculus e Valve fizeram para esse mercado.</p>

Será necessário definir os seguintes estilos para os elementos HTML:


  • Parágrafo:

    • Tamanho da fonte de 18 pixels;
    • Família da fonte Verdana;

  • Elemento de marcação (span):

    • Negrito

Seguindo a ordem em que os estilos foram declarados, preencha as lacunas para que o código CSS corresponda aos estilos acima:


Código CSS

p {

font-size: 18px;

font-family: Verdana;

}

span {

font-weight: bold;

}
