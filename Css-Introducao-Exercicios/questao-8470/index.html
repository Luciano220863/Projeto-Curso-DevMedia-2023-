Estilizando o elemento h2

Alguns estilos foram definidos para o elemento <h2> (subtítulo) de uma página:


  • Tamanho de fonte: 32 pixels;
  • Família de fonte: Impact;
  • Alinhamento do texto à direita
  • Linha sobre o texto (oposto ao sublinhado);

Código HTML

h2 {

font-size: 32px;

font-family: Impact;

text-align: right;

text-decoration: overline;

}
