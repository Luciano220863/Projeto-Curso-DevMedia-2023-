Questão - 8446

Completando os estilos para o link


  • Veja os estilos a seguir:

    • Família de fonte "Impact"
    • Fonte em negrito
    • cor de fundo rgb (244, 253, 17)

Complete o código CSS abaixo para que esses estilos sejam aplicados no elemento link:

a {

font-family: Impact;

font-weight: bold;

background-color: rgb(244, 253, 17);

}
