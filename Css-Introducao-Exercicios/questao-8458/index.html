Completando o código CSS

Veja abaixo os elementos e seus estilos:

  • Parágrafo:

    • Cor de texto #787878;
    • Alinhamento do texto justificado

  • Link:

    • Remoção do sublinhado
    • tamanho de fonte de 16 pixels

Complete o código CSS abaixo:

<style>

paragrafo {

color: #787878;

text-align: justify;

}

link {

text-decoration: none;

font-size: 16px;

}

</style>
