Identificando os padrões de cor

No arquivo estilo.css vemos que foram aplicados alguns padrões de cor nas propriedades color e background-color:


h1 {

color: rgb(244,244,25);

}


p {

color: blue;

text-align: center;

}


a {

color: #454545;

background-color: #c1c1c1;

}


Note que o padrão de cor nominal foi utilizado para definir a cor texto do parágrafo
