Estilos para o link

Veja os itens que devem ser implementados no link:

    • Família de fonte Arial
    • Fonte em negrito
    • Cor de texto azul no padrão hexadecimal
    • Cor de fundo cinza no padrão hexadecimal

Marque qual das alternativas corresponde exatamente aos itens listados acima.

<style>

a {

font-family: Arial;

font-weight: bold;

color: #1122ff;

background-color: #c1c1c1;

}

</style>
