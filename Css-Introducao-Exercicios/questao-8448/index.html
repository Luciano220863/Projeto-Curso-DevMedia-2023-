Estilos para h3 e h4

  • Elemento h3:

    • Cor de texto: #454545
    • Família de Fonte Arial

  • Elemento h4:

    • Cor do texto: #787878
    • Tamanho da fonte: 16 pixels

    • Complete o código CSS para que os estilos atendam aos elementos acima.


OBS.: Siga a ordem em que os estilos foram definidos.

h3 {

color: #454545;

font-family: Aria

}


h4 {

color: #787878;

font-size: 16px;

}
