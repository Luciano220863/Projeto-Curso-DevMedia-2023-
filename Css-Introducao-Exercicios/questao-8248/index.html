Será necessário criar uns estilos para o título no conteúdo da página. Abaixo segue uma lista desses estilos:

  • Tamanho da fonte: 28 pixels
  • Família da fonte: Verdana
  • Cor de texto do título: azul

Abaixo a alternativa que corresponde à estilização proposta acima para o título.



h1 {

color: blue;

font-size: 28px;

font-family: Verdana;
