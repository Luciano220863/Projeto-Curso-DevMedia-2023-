Estilizando o título e subtítulo


temos dois elementos HTML dentro de uma div:

<div>

<h1>Novo filme da DC a caminho</h1>

<h2>Rumores de que um novo filme sobre um misterioso personagem estava sendo gravado nas últimas semanas</h2>

</div>



Então é certo afirmar que conforme a codificação abaixo teremos como resultado; A cor de texto do subtítulo (h2) será azul:

div {

font-family: Tahoma

font-size: 28px;

color: black;

}


h1 {

text-align: center;

}


h2 {

font-size: 24px;

color: blue;

}
