1. Introdução

Para mudar a aparência de uma página web é necessário utilizar uma linguagem de estilo chamada CSS (Cascading Style Sheets - Folha de Estilo em Cascata).
Com ela podemos mudar o visual dos elementos que compõem um documento HTML.
