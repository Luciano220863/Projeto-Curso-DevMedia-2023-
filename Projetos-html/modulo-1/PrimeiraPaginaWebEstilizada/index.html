Questão 7021 - Editar Perfil

Complete o código abaixo com base nas instruções:

  • A página tem o título Editar Perfil;
  • A página tem a descrição "dados do perfil";
  • A página não deve ser exibida nos resultados de busca e seus links devem ser privados;
  • A página deve lidar com textos acentuados.

Respostas

...
<head>

<title>Editar perfil</title>

<meta name="description" content="dados do perfil">

<meta name="robots" content="noindex, nofollow">

<meta charset="UTF-8">

</head>
...

Questão 7040 - Notícias de filmes

Complete as lacunas abaixo seguindo as instruções:

  • A página deve ter um título;
  • A página deve ter uma descrição;
  • A página deve conseguir lidar com conteúdo acentuado;
  • A página deve adicionar um estilo CSS adicionado corretamente ao cabeçalho da página.

    • Respostas

<head>

<title>Notícias sobre filmes</title>

<meta name="description" content="Página de filmes">

<meta charset="UTF-8">

<link rel="stylesheet" href="style.css">

<style>
p { color: #ff0000; }
</style>
</head>
