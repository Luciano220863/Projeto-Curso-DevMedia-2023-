Questão 7026 Comentários do curso

Complete as lacunas abaixo corretamente:

  • A página deve ter seus links privados dos buscadores;
  • Não é necessário remover a página dos resultados de busca;
  • A página deve poder lidar com caracteres acentuados através do encode UTF-8.

Respostas

...

<!doctype HTML>

<html>

<head>

<title>Comentários do curso</title>

<meta name="robots" content="nofollow">

<meta charset="UTF-8">

<style>

h1 {

color: blue;

}

p {

color: green;

}

</style>

</head>

<body>

<h1>Comentários do curso</h1>

Muito bom, gostei do conteúdo.

legal, agora consegui aprender.

</body>

</html>

Questão 7047 - Times de basquete

  • Complete o código abaixo com as tags de cabeçalho corretas:

    • A página tem o título "Times de basquete";
    • A página tem a descrição "Times de basquete";
    • A página deve se ajustar a qualquer tela e ter o zoom inicial padrão (1)

Respostas

<!doctype HTML>

<html>

<head>

<title>Times de basquete</title>

<meta name="viewport" content="width=device-width, initial-scale=1.0">

</head>

<img src="goldenstate.png">Warriors</p>

</div>

</body>

</html>
