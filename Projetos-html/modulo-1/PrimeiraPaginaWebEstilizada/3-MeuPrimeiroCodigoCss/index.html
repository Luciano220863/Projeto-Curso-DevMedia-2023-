No Código 1 vemos o código que foi utilizado no flow.

Código 1. Código HTML

<!DOCTYPE html>

<html>

<head>

<meta charset="UTF-8">

<title>Estrutura CSS</title>

<style>

h1 {

font-size: 48px;

color: blue;

}

p {

font-size: 20px;

}

</style>

</head>

<body>

Cada propriedade no CSS nos permite alterar uma característica de um elemento no HTML. Podemos ver na Figura 1 a propriedade font-size, que altera o tamanho da fonte de um elemento de texto.


Um código CSS deve estar com a sintaxe correta e também ficar dentro da tag style para que os estilos sejam aplicados de forma correta
